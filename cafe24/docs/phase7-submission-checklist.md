로드맵 원문 7-3 "제출 물품 체크리스트"의 5개 항목을 다음과 같이 현황 정리합니다:

### ✅ #1: 적용 URL (스토어프론트)

**상태:** ⏳ 보류 (실 카페24 테스트몰 필요)

**현재:** 준비되지 않음

**제출 시점:**
1. Phase 1 가이드(`docs/phase1-setup-guide.md`) 참고하여 카페24 테스트몰 생성
2. 스마트디자인 편집창에 다음 3개 파일을 삽입:
   - `cafe24/src/detail-option.html` → `/product/detail.html`에 마크업 추가
   - `cafe24/src/option.css` → 스타일 적용
   - `cafe24/src/option.js` + `cafe24/config/optionConfig.js` → 로직 및 설정 적용
3. 스토어프론트 URL 확보 (형식: `https://{몰도메인}.cafe24.com/product/{상품번호}`)
4. 이 체크리스트에 URL 기입

**예시:**
```
적용 URL: https://test-option-ui.cafe24.com/product/12345
```

### ✅ #5: 화면 녹화 (GIF/영상 링크 또는 파일)

**상태:** ⏳ 보류 (실 카페24 스토어 필요)

**필요한 4개 녹화:**
1. **#2 동기화**: 커스텀 UI 카드 클릭 → 기본 옵션도 함께 변경 (5~10초)
2. **#3 중복방지**: 같은 카드 재클릭 → 경고 토스트 (5~8초)
3. **#4 횟수제한**: 그룹 가득 시 추가 불가 & 경고 (8~12초)
4. **#7 활성화상태**: 선택/미선택/비활성화 상태 시각적 구분 (8~10초)

**시나리오 및 촬영 팁:**
- 상세 가이드: `docs/phase6-recording-plan.md` 참고
- 포맷: GIF < 10MB 또는 MP4 < 50MB
- 도구: Chrome 개발자도구 Device Mode + OBS Studio 또는 ScreenFlow

**제출 시:**
- 로컬 파일 첨부, 또는
- YouTube/Google Drive 링크 제공

## 📋 실 스토어 준비 후 순서

### Step 1: Phase 1 환경 준비 (1시간)
- `docs/phase1-setup-guide.md` 참고하여 카페24 테스트몰 생성
- 상품 등록, 옵션 설정 (8개)
- 스마트디자인 편집창 진입

### Step 2: 파일 적용 (0.5시간)
- 다음 3개 파일을 스마트디자인 `/product/detail.html`에 삽입:
  1. `cafe24/src/detail-option.html` (마크업)
  2. `cafe24/src/option.css` (스타일)
  3. `cafe24/src/option.js` + `cafe24/config/optionConfig.js` (로직+설정)
- 스토어프론트에서 정상 작동 확인

### Step 3: QA 재검증 (1시간)
- `docs/phase6-qa-checklist.md`의 ⚠️ 항목 재검증:
  - #2 동기화: 기본 옵션과 양방향 동시 변경 확인
  - #6 폼 제출: 장바구니 담기, 바로구매 정상 작동 확인
- 체크리스트 항목 완료 표기

### Step 4: 화면 녹화 (0.5~1시간)
- `docs/phase6-recording-plan.md`의 4개 시나리오 촬영
- 각 녹화 파일 저장 (GIF 또는 MP4)

### Step 5: 이 체크리스트 최종 작성 (0.5시간)
- #1 적용 URL 기입
- #5 화면 녹화 링크/파일명 기입
- #2~#4 완료 표기

### Step 6: 최종 제출
- GitHub 저장소 OR zip 파일
- 이 체크리스트 + 관련 링크
- 테스트 안내받은 이메일로 회신
